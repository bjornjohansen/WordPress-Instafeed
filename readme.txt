Stream of photos from Instagram on your WordPress site

== Description ==

Fetches the latest instagrams from either a specified Instagram username or a tag.

Works on https.


== Installation ==
1. Download plugin
2. Upload the plugin through the 'Plugins' > 'Add new' > 'Upload' page in WordPress,
3. Activate the plugin through the 'Plugins' menu in WordPress.
